This repo contains some utility scripts for daily use.

# Installation
1. Clone repository to ~/scripts
1. Add the following to `.zshrc` or `.bashrc`

```.zshrc
alias asp='sh ~/scripts/awscli-select-profile.sh'
alias gci='sh ~/scripts/git-checkout-interactive.sh'
alias gbdi='sh ~/scripts/git-delete-branch.sh'
alias rebase='sh ~/scripts/git-rebase-mainawscli.sh'
```
